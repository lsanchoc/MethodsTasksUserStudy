Organization
-------------

The software is organized into four folders, one for each method; that is:

> Proyecto_Visualizacion_Agglomeration: contains the code and data for the agglomeration method.
> Proyecto_Visualizacion_Animation: contains the code and data for the animation method.
> Proyecto_Visualizacion_Edge: contains the code and data for the edge-drawing method.
> Proyecto_Visualizacion_Matrix: contains the code and data for the matrix method.

How to run the software?
-------------------------
1. Download the folders and files from the SoftwareAndData folder.
2. Go into any of the method folders; for instance, go into "Proyecto_Visualizacion_Edge" and open the "index.html" file with a web browser (Mozila Firefox or Google Chrome). 
3. A web browser with the implementation of the four methods will be displayed.
